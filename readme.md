# Ran-chess

Ran-chess is Chess with random figures position except the Pawn.

### Installation

This project use C++ and SFML library.
Environment: Linux.

```sh
$ sudo apt-get install libsfml-dev
```

### Run 

```sh
$ g++ -c main.cpp
$ g++ main.o -o sfml-app -lsfml-graphics -lsfml-window -lsfml-system 
$ ./sfml-app
```
